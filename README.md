# deckofcards-api
Demo Test framework project for Deckofcardsapi.com website using Java, TestNG and Rest Assured

1.) After cloning the project from github, launch the command interface (CMD) and change the directory to runenv folder. e.g. CD C:\Automation\deckofcards-api\runenv.

2.) Run java -cp ".\*;C:\<Project Folder Name>\target\test-classes" org.testng.TestNG C:\<Project Folder Name>\testng.xml

3.) You should see the following output:
	===============================================
	Deck Of Cards - End to End Testing
	Total tests run: 4, Passes: 4, Failures: 0, Skips: 0
	===============================================	
	
4.) Open C:\<Project Folder Name>\runenv\test-output\emailable-report.html to see a detailed report.

N.B. Please take a look at the C:\Automation\deckofcards-api\screenshots folder to see how your reports should look like.
